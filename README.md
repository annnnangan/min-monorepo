# min-monorepo

Minimal monorepo template for a React TypeScript frontend and Python FastAPI backend web application using Cursor.

## Overview

This starter includes:
- **Backend**: Pre-configured FastAPI app (Python 3.13+)
- **Frontend**: Created on-demand via Cursor command (React + TypeScript + Vite)

The frontend is generated when you run the Cursor command, so you always get the latest Vite/React versions.

## Quick Start

### 1) Backend
```bash
cd backend
uv sync
uv run main.py
```
Backend runs on `http://localhost:8000`.

### 2) Frontend (initialize via Cursor)
- Run Cursor command `@init-frontend` from the repo root.
  - Detects package manager (prefers `bun`, falls back to `npm`)
  - Scaffolds Vite + React + TypeScript
  - Configures `/api` proxy to the backend
  - Adds example API integration

After it creates `frontend/`:
```bash
cd frontend
npm install  # or bun install
npm run dev  # or bun run dev
```
Frontend runs on `http://localhost:5173`.

## Features

### Backend (included)
- FastAPI with automatic OpenAPI docs at `/openapi.json`
- Pydantic settings, CORS pre-configured
- Example endpoints: `GET /api/hello`, `GET /health`
- Environment-based config via `.env`

### Frontend (created via command)
- React 19+ with TypeScript
- Vite dev/build, ESLint configured
- `/api` proxy to `http://localhost:8000`
- Example fetch to `/api/hello` shown in `App.tsx`

## Development Workflow
1) Start backend: `cd backend && uv run main.py`
2) Initialize frontend (first time only): run `@init-frontend`
3) Start frontend: `cd frontend && npm run dev` (or `bun run dev`)
